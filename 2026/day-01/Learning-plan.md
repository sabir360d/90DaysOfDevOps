## Understanding of DevOps & Cloud Engineering
DevOps is a **culture, mindset & set of practices** that enables faster, reliable software delivery through automation, collaboration, and shared ownership between development and operations.
Cloud Engineering provides **scalable, on-demand infrastructure** that supports modern application deployment, monitoring, and resilience.
Together, DevOps and Cloud emphasize automation, continuous delivery, observability, and end-to-end system ownership.

## Why I Am Learning DevOps & Cloud
I am pursuing DevOps and Cloud to:
- Upskill for **career growth and role elevation**
- Build strong hands-on expertise in **Linux, CI/CD, containers, and cloud**
- Understand **how systems work end-to-end**, not just individual tools
- Move into a modern, high-impact engineering role

Linux is a key focus because most production workloads run on Linux, making it foundational for DevOps and SRE roles.

## 90-Day Goals
By the end of 90 days, I will:
1. Deploy a **containerized application on AWS** using Docker and Kubernetes  
2. Build a **production-ready CI/CD pipeline** using GitHub Actions  
3. Troubleshoot **Linux, Docker, and Kubernetes issues** with confidence
